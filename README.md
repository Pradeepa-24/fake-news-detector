# AI-Powered Fake News Detection System

## Overview

This project uses Machine Learning and Natural Language Processing (NLP) to classify news articles as Real News or Fake News.

## Features

- Fake News Detection
- Real News Detection
- Confidence Score Prediction
- Flask Web Application
- Interactive User Interface

## Technologies Used

- Python
- Flask
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression
- HTML
- CSS

## Dataset

- Fake.csv
- True.csv

## Model Accuracy

98.58%

## How to Run

1. Train the model

```bash
python train.py
```

2. Run the application

```bash
python app.py
```

3. Open in browser

```text
http://127.0.0.1:5000
```
